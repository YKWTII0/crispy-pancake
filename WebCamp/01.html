HTML: Hyper Text Markup Language

HTML의 특징

  • Hyper Text(하이퍼텍스트)는 링크로 연결된 텍스트 페이지
  • Markup Language(마크업 언어)는 태그 등을 이용해서 문서를 작성하도록 도와주는 언어
  • HTML은 웹의 기본. 마크업 언어일 뿐 아니라, 웹 애플리케이션을 구성하는 다양한 스펙들이 존재.
  • 확장자는 .html

HTML 버전 이야기

HTML 1.0, HTML 2.0 ... 지속적으로 버전업.
최신 스펙으로 알려져있던 HTML5는 폐기, HTML Living Standard가 현재 최신 버전.
